Création d'un Pixel War partie Front

  • Languages: HTML/CSS JS
  • Date du projet: 28 avril 2024

Vous êtes créatif, vous aimez les challenges et la liberté de créer, alors la guerre des pixels est faites pour vous, rejoignez une équipe parmi les quatre et commençait a vous battre contre d'autres poseur de pixel afin de réaliser la plus belle oeuvre. Durant ce projet nous avons du étudiez la documentation de l'API existante fournie avec Swagger, comprendre les différentes routes disponibles, les paramètres acceptés et les réponses renvoyées par l'API. Nous avons conçue une interface utilisateur conviviale pour le jeu avec plusieurs thèmes de couleur que vous pouvez changer aléatoirement.
